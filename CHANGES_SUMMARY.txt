├─ Accept: JSON with { "urls": [...] }
├─ Returns: DocumentsResponse (201 Created)
├─ Auth: Access or Onboarding token
└─ Example:

     -H "Authorization: Bearer TOKEN" \
     -H "Content-Type: application/json" \
     -d '{"urls": ["https://example.com"]}'

┌────────────────────────────────────────────────────────────────────────────────┐
│ PERFORMANCE CHARACTERISTICS                                                     │
└────────────────────────────────────────────────────────────────────────────────┘

PDF Upload:
  • Time: ~1-3 seconds per file (Cloudinary dependent)
  • Blocking: Sync operation
  • Memory: File stored in memory

Web Scraping:
  • Time: ~5-30 seconds per URL (website dependent)
  • Blocking: Async crawl, sync PDF generation
  • Memory: Content + PDF in memory
  • Timeout: 30 seconds (configurable)

Database Operations:
  • Document list: < 1 second for 100 documents
  • Quota check: < 100ms
  • Index: tenant_id, source, status

┌────────────────────────────────────────────────────────────────────────────────┐
│ QUICK START                                                                     │
└────────────────────────────────────────────────────────────────────────────────┘

Install:
  pip install -r requirements.txt

Migrate DB:
  psql -d securerag < migrations/add_documents_table.sql

Configure:
  CLOUDINARY_CLOUD_NAME=your_name
  CLOUDINARY_API_KEY=your_key
  CLOUDINARY_API_SECRET=your_secret
  CRAWL4AI_ENABLED=true

Upload PDF:
  curl -X POST http://localhost:8000/api/v1/documents/upload \
    -H "Authorization: Bearer TOKEN" \
    -F "files=@document.pdf"

Scrape Website:
  curl -X POST http://localhost:8000/api/v1/documents/scrape \
    -H "Authorization: Bearer TOKEN" \
    -H "Content-Type: application/json" \
    -d '{"urls": ["https://example.com"]}'

List Documents:
  curl -X GET http://localhost:8000/api/v1/documents/ \
    -H "Authorization: Bearer TOKEN"
